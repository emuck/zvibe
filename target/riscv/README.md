# ZVibe RISC-V Target

This directory contains the RISC-V implementation of the ZVibe Z-machine interpreter.

## Active Development

All active development is in the **`serv_zvibe/`** directory:

```bash
cd serv_zvibe/
```

See [`serv_zvibe/README.md`](serv_zvibe/README.md) for:
- Multi-board architecture overview
- Supported FPGA boards (MAX10 08 Eval, Arty S7-50)
- Build instructions
- Getting started guide
- Documentation links

## Directory Contents

- `serv_zvibe/` — RISC-V SoC implementation (SERV CPU, multi-board)
